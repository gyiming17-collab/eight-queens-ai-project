## 核心实现思路
### 算法原理：回溯法
八皇后问题要求在8×8棋盘上放置8个皇后，使得任意两个皇后不在同一行、列或对角线上。核心逻辑：
1. **逐行放置**：每行仅放置1个皇后，通过数组`board[row] = col`记录每行皇后的列位置；
2. **冲突检查**：通过`is_safe()`函数验证当前位置是否满足：
   - 列冲突：同列无其他皇后；
   - 对角线冲突：行差绝对值 ≠ 列差绝对值；
3. **递归回溯**：若当前行所有列均不合法，则回退到上一行，尝试下一个列位置；若所有行放置完成，则记录为有效解。

## Bug引入与调试
### 1. 故意引入Bug
修改`src/eight_queens.py`中`is_safe()`函数，删除对角线冲突判断逻辑：
```python
# Bug版本（仅保留列冲突判断）
def is_safe(row: int, col: int) -> bool:
    for i in range(row):
        if board[i] == col:  # 仅检查列冲突，删除对角线判断
            return False
    return True
```

### 2. Bug现象验证
运行程序后会出现典型错误：
- 解数量从**92个**暴增至**40320个**（8! 全排列结果）；
- 第一个解的棋盘布局中，皇后全部落在主对角线上（明显对角线冲突）；
- 单元测试`test_8_queens_solution_count`断言失败（`40320 != 92`）。

### 3. Bug修复方案
恢复对角线冲突判断逻辑，修正`is_safe()`函数：
```python
# 正确版本（完整冲突检查）
def is_safe(row: int, col: int) -> bool:
    for i in range(row):
        # 列冲突 OR 对角线冲突
        if board[i] == col or abs(board[i] - col) == abs(i - row):
            return False
    return True
```

### 4. 修复验证
重新运行程序和单元测试，确认：
- 解数量回归92个；
- 棋盘布局无对角线冲突；
- 所有单元测试用例通过。
